# Artifact Builder

Goal:
Build or improve the canonical operator facing artifact for one alert type.

Steps:
1. Pick one alert type.
2. Define the artifact sections:
   - Summary
   - Why this fired
   - Evidence
   - Estimated impact
   - Recommended next steps
3. Rewrite language to be operator friendly.
4. Implement the artifact view.
5. Add a test dataset and verify output.
6. Summarize how an operator would use this artifact.
